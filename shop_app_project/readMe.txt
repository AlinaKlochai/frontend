Реализовать страницу с продуктами
документация:
https://dummyjson.com/docs/products
1) на странице отобразить все продукты (верстка на ваше усмотрение)
2) * На странице отобразить как продукты так и  категории и при клике на категорию отображать продукты только данной категории
3) Реализовать страницу добвления товара на странице отобразить форму с необъодимыми полями и кнопкой отправить
   p.s. - публичное апи https://dummyjson.com/docs/products не дает возможность реального сохранения новых записей в базу она делает лишь имитацию сохраненния данных...
4) Реализовать страницу отображения информации по одному товару - на данной странице реализовать карусель с картинками - добавить кнопку удалить товар 
   ( данное апи не дает реально удалить товар - а делает лишь эмитацию удаления товара)
5) На главной странице с товарами и категориями - реализовать поиск по названию товара т.е. на странице должен быть input с типом текст и при вводе в 
   него значения и нажатия кнопки найти - осуществляется поиск товара по названию
примеры url:
https://dummyjson.com/products/search?q=phone
https://dummyjson.com/products/search?q=laptop
https://dummyjson.com/products/search?q=lap
